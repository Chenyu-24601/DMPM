# 上传文件到团队

我来帮你上传文件到团队仓库。

## 第一步：确认文件路径

请告诉我你要上传的文件路径。

**示例：**
- `DMPM/skills/my-skill/SKILL.md`
- `DMPM/skills/product-prototype/references/template.md`
- `DMPM/docs/new-guide.md`

**要求：**
- 文件必须在 `DMPM/` 目录下
- 可以使用相对路径或绝对路径

## 第二步：执行上传

确认文件路径后，我会运行上传命令。

**上传流程：**
1. 验证文件存在
2. 检查文件是否在 DMPM/ 目录
3. 显示文件信息并确认
4. 执行上传
5. 通知上传结果
